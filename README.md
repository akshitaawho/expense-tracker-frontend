# Expense Tracker Frontend

A responsive frontend for the Expense Tracker application built with Next.js and TypeScript.

---

## Live Demo

https://expense-tracker-frontend-rtyt.onrender.com

---

## Features

- Add expenses
- Edit expenses
- Delete expenses
- Search expenses
- Filter by category
- Sort expenses
- Dashboard summary
- Responsive UI
- Connects to deployed Spring Boot backend

---

## Tech Stack

- Next.js
- React
- TypeScript
- CSS

---

## Backend

The backend is built using Spring Boot and deployed on Render.

API Documentation:

https://expense-tracker-api-q93i.onrender.com/swagger-ui/index.html
